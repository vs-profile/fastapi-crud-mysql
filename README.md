# FastAPI CRUD API with MySQL

A REST API built using FastAPI, SQLModel, SQLAlchemy, and MySQL.

## Features

- Create Customer
- Get All Customers
- Get Customer by ID
- Update Customer
- Delete Customer

## Tech Stack

- FastAPI
- SQLModel
- SQLAlchemy
- MySQL
- PyMySQL
- Uvicorn

## Run

```bash
pip install -r requirements.txt
uvicorn main:app --reload
